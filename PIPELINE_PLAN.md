### Phase 1: Data Utilities & Converters (Day 1-2)

**Goal:** Convert all annotation formats to unified YOLO format, build depth utilities.

#### 1.1 `scripts/convert_cityscapes_labels.py`
Convert Cityscapes `gtFine_instanceIds.png` → YOLO bbox `.txt` files.

```
Input:  cityscapes/gtFine_trainvaltest/gtFine/train/{city}/{base}_gtFine_instanceIds.png
Output: cityscapes/labels/train/{city}/{base}.txt  (YOLO format: class cx cy w h)

Cityscapes instance ID → class mapping (8 detection classes):
  24: person    → 0
  25: rider     → 1
  26: car       → 2
  27: truck     → 3
  28: bus       → 4
  31: train     → 5
  32: motorcycle→ 6
  33: bicycle   → 7

Algorithm:
  1. Load instanceIds.png
  2. For each unique instance ID (excluding 0/255):
     a. Extract class ID = instanceId // 1000
     b. Create binary mask for this instance
     c. Find bounding box (min_x, min_y, max_x, max_y)
     d. Convert to YOLO format: cx, cy, w, h (normalized 0-1)
  3. Write to .txt file
```

#### 1.2 `scripts/convert_acdc_labels.py`
Convert ACDC COCO JSON → YOLO bbox `.txt` files.

```
Input:  gt_detection_trainval/gt_detection/fog/instancesonly_fog_train_gt_detection.json
Output: acdc/labels/train/{sequence}/{frame}.txt

COCO format: {image_id, category_id, bbox: [x, y, w, h]}
YOLO format: class_id cx cy w h (normalized)

Categories (already 0-7, matching Cityscapes):
  0: person, 1: rider, 2: car, 3: truck, 4: bus, 5: train, 6: motorcycle, 7: bicycle
```

#### 1.3 `src/data/depth_utils.py`
Disparity → metric depth conversion.

```python
FOCAL_LENGTH = 2262.52  # pixels
BASELINE = 0.209313     # meters

def disparity_to_depth(disparity_png_path) -> np.ndarray:
    """Cityscapes disparity PNG → metric depth in meters."""
    disp = np.array(Image.open(path), dtype=np.float32)
    # Cityscapes stores disparity * 128 (fixed point)
    disp = disp / 128.0
    depth = (FOCAL_LENGTH * BASELINE) / (disp + 1e-8)
    depth[disp <= 0] = 0  # Invalid pixels
    return np.clip(depth, 0, 80.0)
```

```python
def wrdnet_collate_fn(batch):
    """Collate function that handles variable-length bbox lists."""
    images = torch.stack([b['image'] for b in batch])
    clear_gts = torch.stack([b['clear_gt'] for b in batch]) if 'clear_gt' in batch[0] else None
    depth_gts = torch.stack([b['depth_gt'] for b in batch]) if 'depth_gt' in batch[0] else None
    bboxes = [b['bboxes'] for b in batch]  # List of [N, 5] tensors
    return {'image': images, 'clear_gt': clear_gts, 'depth_gt': depth_gts, 'bboxes': bboxes}
```

### Phase 3: Data Pipeline Assembly (Day 3)

**Goal:** Wire up dataloaders for all training modes.

#### 3.1 `src/data/dataset.py` (Rewrite `build_dataloaders`)

```python
def build_dataloaders(config):
    """
    Build dataloaders based on config.

    Modes:
      1. Supervised only: Cityscapes (foggy + clear + depth + labels)
      2. Supervised + DA: Cityscapes + ACDC/Zurich (paired batches)
      3. ACDC only: Real fog training
    """
    # Synthetic (labeled): Cityscapes
    synth_train = FoggyCityscapesDataset(
        root='data/cityscapes',
        split='train',
        fog_density='0.02',  # Use densest fog
        config=config,
    )

    # Real (labeled): ACDC
    real_labeled = ACDCDataset(
        root='data/rgb_anon_trainvaltest',
        split='train',
        labels_dir='data/acdc/labels',
        config=config,
    )

    # Real (unlabeled): Foggy Zurich
    real_unlabeled = FoggyZurichDataset(
        root='data/Foggy_Zurich',
        config=config,
    )

    # Validation: ACDC val (real fog with labels)
    val_dataset = ACDCDataset(
        root='data/rgb_anon_trainvaltest',
        split='val',
        labels_dir='data/acdc/labels',
        config=config,
    )

    # Combine for DA training
    if config.use_domain_adaptation:
        # Combine ACDC labeled + Zurich unlabeled as "real" domain
        real_combined = ConcatDataset([real_labeled, real_unlabeled])
        train_dataset = PairedDADataset(synth_train, real_combined)
    else:
        # Just synthetic
        train_dataset = synth_train

    train_loader = DataLoader(train_dataset, batch_size=config.batch_size,
                              shuffle=True, num_workers=4, collate_fn=wrdnet_collate_fn)
    val_loader = DataLoader(val_dataset, batch_size=config.batch_size,
                            shuffle=False, num_workers=4, collate_fn=wrdnet_collate_fn)

    return train_loader, val_loader
```

### Phase 4: YOLOv11s Integration Fix (Day 4)

**Goal:** Fix backbone feature extraction and neck/head injection.

#### 4.1 Fix `get_backbone_features`

**Problem:** Uses `torch.no_grad()` — breaks gradient flow for training.
**Problem:** Re-registers hooks every call — slow and memory leak.

**Fix:**
```python
def __init__(self):
    # Register hooks ONCE in __init__
    self._features_cache = {}
    for name, idx in {'P3': 6, 'P4': 8, 'P5': 10}.items():
        self.model.model[idx].register_forward_hook(self._hook_fn(name))

def get_backbone_features(self, x):
    # Run forward through backbone only (layers 0-10)
    # NO torch.no_grad()
    for i in range(11):  # backbone = layers 0-10
        x = self.model.model[i](x)
    return dict(self._features_cache)  # Return cached features
```

#### 4.3 Integrate Real YOLO Detection Loss

**Problem:** `losses['det'] = torch.tensor(0.0)` — no actual detection loss.

**Fix:** Use ultralytics' built-in `v8DetectionLoss`:

```python
class WRDNetLoss(nn.Module):
    def __init__(self, config):
        # ...
        # YOLOv11s detection loss (same as v8)
        self.yolo_loss = v8DetectionLoss(
            model=self._get_yolo_model_for_loss()

    def forward(self, outputs, batch):
        # Detection loss
        if 'detections_s' in outputs and 'bboxes' in batch:
            # Convert bboxes to ultralytics format
            targets = self._prepare_yolo_targets(batch['bboxes'])
            losses['det'] = self.yolo_loss(outputs['detections_s'], targets)[0]
```

## Key Design Decisions

1. **Unified label format:** All datasets convert to YOLO format (class_id, cx, cy, w, h) normalized [0, 1]
2. **8 detection classes:** person, rider, car, truck, bus, train, motorcycle, bicycle (matches ACDC)
3. **Fog density:** Default to β=0.02 (densest fog) for main training. Can ablate with 0.005, 0.01
4. **Image size:** 640×640 for all images (YOLO standard)
5. **Depth:** Metric depth in meters, clipped to [0, 80]
6. **Collate:** Custom collate handles variable-length bbox lists
7. **DA pairing:** Synthetic (Cityscapes) paired with real (ACDC + Zurich) for domain adaptation

| Risk | Mitigation |
|---|---|
| YOLO neck injection fails | Start with Approach A (fake backbone), upgrade to B |
| Detection loss NaN | Start with λ_rest=0.1, increase gradually |
| OOM on GPU | Batch size 4, gradient accumulation |
| Data loading slow | num_workers=4, pin_memory=True |
| Bbox conversion errors | Verify with visualization script |
| Depth conversion wrong | Compare with SFSU depth maps |
